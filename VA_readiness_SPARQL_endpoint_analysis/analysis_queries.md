# CLASSES

### used classes queries
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?class ?label WHERE { 
	?s a ?class. 
	OPTIONAL{
		?class rdfs:label ?label.
		OPTIONAL{
			FILTER(LANG(?label)="en")
		}
	}
}

### skos:Concept
